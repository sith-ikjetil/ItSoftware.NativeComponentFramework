# ItSoftware.NativeComponentFramework
License: **GPL-3.0-or-later**  
This is a small native COM Framework with the following COM components:  
* EventLog  
* Guid  
* IniFile  
* Registry  
* String  
* Directory  
* File  
* Path  
* StringStream  
* NetworkShare  
* Network  
* MemoryStream  
* FileStream  
* List  
* Map  
* PriorityQueue  
* AudioCD  
* AudioPlayer  
* Mp3Tag  
* Mp3Encoder  
* WavEncoder  
* AacTag  
* AacEncoder  
* WmaTag  
* WmaEncoder  
   
 The Audio components uses the Microsoft Media Foundation API.
